## Markdown-элементы
### Заголовки (H1–H3)
```markdown
# Заголовок уровня H1
## Заголовок уровня H2
### Заголовок уровня H3
---
### Горизонтальная линия
***
---
### Форматирование текста
**полужирный текст**
*курсивный текст*
~~зачёркнутый текст~~
`моноширинный текст`
### Списки
#### Маркированный список:
- первый пункт
- второй пункт
- третий пункт
#### Нумерованный список:
1. первый пункт
2. второй пункт
3. третий пункт
#### Вложенный список:
- пункт 1
  - подпункт 1.1
  - подпункт 1.2
